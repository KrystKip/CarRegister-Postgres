# CarRegister-Postgres

Small C# console app using PostgreSQL + Entity Framework Core.

## What it does
- Stores gasoline and electric cars  
- Fast CRUD operations  
- CSV import/export  
- Automatic DB migrations  

## How to start
1. Create PostgreSQL DB `carregister_db`
2. Add `appsettings.Local.json` with your connection string
3. Run the project: `dotnet run`

## Tech
- .NET 8  
- EF Core  
- PostgreSQL
